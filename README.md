# js-cpf-validation

Lib to cpf validation

#### Javascript example:
```js
const cpf = require('js-cpf-validation');  
console.log(cpf.isValidCPF("153.856.490-40"))
``` 
Output:
true

#### Typescript example:
```ts 
import { isValidCPF } from 'js-cpf-validation'
console.log(isValidCPF("153.856.490-40")
```

Output:
true
